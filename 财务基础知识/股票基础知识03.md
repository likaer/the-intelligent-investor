## 股息率
 - 公式：股息率 = 每股分红 ÷ 当前股价
 - 角度：站在 投资人角度，衡量你买入这只股票后能拿到多少分红回报。
 - 比喻：好比你把钱存银行，银行给你的利率是多少。

例：股价 20 元，每股分红 1 元
→ 股息率 = 1 ÷ 20 = 5%

## 派息率
 - 公式：派息率 = 每股分红 ÷ 每股收益（EPS）
 - 角度：站在 公司角度，衡量公司赚的钱有多少分给股东。
 - 比喻：好比你发工资，公司发奖金，是把净利润的多少比例拿来分红。

例：每股收益 EPS = 2 元，每股分红 = 1 元
→ 派息率 = 1 ÷ 2 = 50%
意思是：公司赚的钱里，有一半分给股东，另一半留在公司再投资或偿还债务。



## 如何查看这两个指标

1. 首先应该先看一家企业历史财报的每股分红/每股收益，派息率30%-70%（长期可持续）。
低于30%则说明企业很抠搜，高于 70%，容易在市场失去竞争力，可持续发展能力堪忧
2. 基于当前股价计算股息率( 历史派息率x预估的当年度净利润/当前总市值)，我个人倾向于只考虑股息率大于5%的股票，具体多少，取决于企业本身的增长率+企业的股息率换算得到的复利效应两者哪个更高

## chatgpt点评

1. 派息率不要一刀切，按行业细分，比如：
 - 银行：40–60%
 - 公用事业、电信：50–80%
 - 制造/成长型企业：20–40%
2. 股息率 + 增长率 = 最小预期收益率
 - 可以设一个目标，比如：
 - 股息率+增长率≥12%就作为候选股。这样可以定量化标准。
3. 现金流匹配
 - 有些公司利润好看但现金流差，分红可能透支（靠借钱分红）。这时候派息率的判断就要结合现金流。



## 接下来我们以中创智领为例，计算一下中创智领的派息率和股息率

派息率=每股分红/每股收益，

 - 每股收益请参考年报中的利润-每股收益，24年每股收益2.2120，23年每股收益1.8530
 - 每股分红参考每年的公告，中创智领24年10派11.2元， 23年10派8.4元	

2024年派息率计算

 - 分红方案：10股派11.2元
 - 每股派息金额：1.12元 (11.2元 / 10)
 - 每股收益（EPS）：2.2120元
 - 计算公式：派息率 = (每股派息 / 每股收益) * 100%
 - 计算结果：派息率 = (1.12元 / 2.2120元) * 100% ≈ 50.63%

2023年派息率计算

 - 分红方案：10股派8.4元
 - 每股派息金额：0.84元 (8.4元 / 10)
 - 每股收益（EPS）：1.8530元
 - 计算公式：派息率 = (每股派息 / 每股收益) * 100%
 - 计算结果：派息率 = (0.84元 / 1.8530元) * 100% ≈ 45.33%
 
假设25 年派息率为50%， 则25年预估分红方案为

 - 派息率：50%
